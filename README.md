# 📊 Data Analytics & Business Intelligence — Task 2
### Maincrafts Technology Internship Programme

## 📌 Project Title
**Advanced Data Analysis, SQL Joins & Business KPI Dashboard**

## 📖 Project Overview
Task 2 builds on Task 1 by working with relational data — joining
Orders and Customers tables via SQL, calculating business KPIs, and
building a structured Power BI dashboard for management reporting.
Dataset: Superstore Sales (9,994 order lines | 5,009 orders | 793 customers).

## 📂 Files in This Repository

| File | Description |
|------|-------------|
| `sales_data.csv` | Source sales dataset |
| `SQL_QUERY_TASK_2.sql` | SQL INNER JOIN + KPI queries (region, category, segment, discount, monthly trend) |
| `Task_2.pbix` | Power BI dashboard with KPI cards, charts and slicers |
| `Task2_Business_Summary_Report.pdf` | 3-page report: KPI overview, regional/category performance, customer analysis, recommendations |
| `Dashboard_Screenshot.png` | Power BI dashboard preview |
